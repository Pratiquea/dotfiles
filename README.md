### Installing Gnome
  ```
  sudo add-apt-repository ppa:gnome3-team/gnome3-staging -y && \
  sudo add-apt-repository ppa:gnome3-team/gnome3 -y &&\
  sudo apt update && \
  sudo apt dist-upgrade && \
  sudo apt install gnome gnome-shell
  ```
